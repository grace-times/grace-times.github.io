如何阅读财报——三大表解读

发表于 2019-04-19 | 分类于 商业 |

先声明一下，文中若提及的公司，都只是示例，不构成投资参考。

我们在预备篇里面已初步介绍过三大表：利润表、资产负债表、现金流量表，它们的作用分别是：

  • 利润表：赚钱
  • 资产负债表：左边花钱、右边找钱
  • 现金流量表：算钱
    并且在分析企业的各个指标中已经认识到了这三张表，下面我们来详细的来看看这三张表。

利润表：长期稳定、获利能力

概念

这里先来回顾一下“利润表”的一些关键知识点，对它的一些理解误区及其应用。

利润表的目的：让我们知道一家公司在一段时间内是赚钱还是亏钱。，有月报，季报，中报（半年）和年报，其中，年报准备时间最充分，信息披露最详细，可参考度最高。

与个人收入相似，营业收入也要扣除各种成本、费用、税之后才能得到净利润，就像一个漏斗。

利润表的几个注意点

1. 净利润不等同于现金流，利润表是推估的概念。

我们平常开个小饭店或杂货铺，可以一手交钱一手交货，天天收现金。但，大部分公司并不是采用这种现金交易的方式。

【付款周期】大多数情况下为了做更多生意，企业都会给客户一定的付款优惠条件。比如先交一部分定金，后面才结算全部款项，所以这当中就会有一段时间，虽然产品或服务已经提供给了客户，但钱还没到。

【收不到款的不确定因素】可能产品质量问题导致收不到全款，甚至退货；若是下游公司倒闭，货款直接就收不回来。还有的可能赖账之类的，在没有打进银行卡前都不能算在自己真正的收入上。

2. 营业收入高，不一定代表公司就好。

就像一个人收入很高，如果花销很大的话，那么他真正赚到手的钱并没有多少了。

企业营业收入高，也不一定代表它赚钱就多，还要看成本和费用，还记得漏斗吗？以后如果看到媒体说某某企业营收创新高时，我们不必激动，不妨思考一下，企业营收这么多，是否真的赚到钱了？在财务世界中，绝对金额没有太大的意义，充其量代表规模大小而已。

3. 看利润表，不能光看单一指标，要连续看5年的数据。
财报是企业的体检表，就像我们身体健康一样，不大可能一下子由好转不好，或者由不好转好，最具代表性的就是毛利率了。

4. 关注行业对比。
不同的行业有不同的特征，比如超市一般利润都不高，但收现金；餐饮业的毛利率要50%以上；白酒行业做生意的完整周期普遍较长，手里现金要多一些才合适，毛利率会很高等等。

我们不能单看某一企业单方面的变化，比如营业收入方面，我们不仅要看企业营业搜入绝对值的增长，还要看是否高于行业平均水平。只有营业收入增长速度高于行业平均增长速度，才能证明企业的市场份额在扩大。

5. 利润表上的营业外收入不靠谱。
如果一家公司的利润大部分收入来源于营业外收入，而不是靠它的主营业务，这样的公司，其盈利能力不能持续。

利润表不仅能告诉我们一家企业今年赚了还是亏了，还能在区分可持续项目和不可持续项目的基础上，推测企业未来的盈利能力。

如何阅读利润表

利润表的八字箴言：【长期稳定、获利能力】

  • 从毛利率判断是不是一门好生意？和同行比较，毛利率处于什么水平？
  • 从营业利润判断有没有赚钱的真本事！毛利-利润=费用，这家企业的费用率如何，企业有没有规模？是否行业前三甲？都可以从费用率中看到。
  • 看营业额是否高？如果营业额很小的话，其他几项意义也不大。
  • 从净利润看，公司是亏钱还是赚钱？
  • 关注利润结构，更重要的是可持续利润

会计利润与经济利润

为什么要区分一下这两个不同的概念呢？因为资金的机会成本，一家公司，投资A项目能赚1000万，投资B项目能赚2000万，现在，这家公司投了A项目，那么，它是赚了还是赔了？站在会计利润的角度，它是赚了1000万的，但是站在经济利润的角度，它是亏了1000万的，因为它原本投资B项目可以赚2000万现在只赚了1000万，然而，机会成本往往是很多人看不见的，且机会成本是浮动变化的，但，我们仍然要尽可能去看到，因为近似的正确好过精确的错误，经济利润比会计利润更能真实的反应公司的经营水平和价值创造能力。

资产负债表

概念

资产负债表反映了一家企业的家底和财务状况。如果说”利润表”给企业一段时间的经营状况拍了一段视频的话，”资产负债表”则是给企业某一天的资产和负债情况拍了一张照片。它表示的是具体某一天一家公司的资产有多少、负债有多少，是一个定量的概念。

资产负债表的左边：

  • 企业的钱花到哪里去了？
  • 哪些还是钱（手上的现金）？哪些变成了应收帐款？哪些变成了原材料、产品？哪些变成了厂房、设备？哪些变成了无形资产（如专利）？

资产负债表的右边：

  • 企业的钱从哪里找来的？
  • 哪些是股东投入的？哪些是从银行借来的？哪些是欠供应商的？

我们得到一个恒等式：
资产=负债+股东权益，即A=D+E

资产负债表上的科目说明

1. 现金
对于企业而言，能在90天内提现的，都属于现金。

2. 存货
存货多是好还是不好要看行业，不要想当然的就认为不好。食品、衣服这类存货太多表示市场越来越小，但白酒、钻石这类，存货则是越多越值钱。

3. 短期与长期
小于等于1年的就叫做短期资产/短期负债。比如一笔五年期的银行贷款还有10个月就到期了，即“一年到期的五年期银行贷款”，因此算作企业的短期负债。

4. 资产与负债按照流动性摆放
资产负债表上的科目是按照资产/负债的流动性来摆放的，容易变现的资产在上边，不好马上变现的资产在下面；短期负债在上面，长期负债在下面。

流动性越高的科目折价率越低，流动性越低的项目，折价率也越高。

无论是企业还是个人，要学会以长支长，千万不能犯以短支长的错误。

如何阅读资产负债表

如果说利润表是企业的外部表现，那么资产负债表则是一家企业的内部支撑。企业的资产状况，财务结构，经营能力，偿债能力在这张表中都有明显的提现。

  • 现金比例有多少？
  • 有无收不回来的呆帐？
  • 有无一直躺在仓库的呆料？
  • 公司整体翻桌率如何？
  • 那根棒子的位置是否合理？与同行业的其他公司比较呢？
  • 股东是否会持续投资？

资产与费用

假如一家蛋糕店接了一个特殊订单，为此买了一批特殊形状的模具。蛋糕做完之后，剩下30个特殊模具。问题来了：这30个剩余模具是资产还是费用呢？

如果把这些剩余模具错误认定为资产，对蛋糕店会有什么影响呢？
首先，把费用错当成资产，当期利润就会增加。蛋糕店就要缴纳更多的税；
其次，这些模具收着又无法卖出去，折不了现。

互联网时代，研发的失败率那么高，那么，研发支出是资产还是费用？
研发是企业核心竞争力，研发支出在未来能够为企业创造巨大的经济利益。所以在那些财务高手眼中，研发支出并不是费用，而是资产。

只有那些未来能够给企业创造利益的经济资源，才是资产。
由于会计准则的局限性，会导致财务报表中认定的资产和企业实际拥有的资源之间不完全对应。

由于无形资产的无法估量，不能量化的资源，会计准则目前就不允许它出现在报表中。所以，企业所拥有的资源，远远大于财务报表中记录的资产。这其实是财务报告现在面临的一个挑战，那就是怎么量化和表征企业的无形资产。

现金流量表：真假立判、生死存亡

公司靠利润发展，靠现金流生存。

概念

现金流量表是一张现金进进出出的报表，只要有任何现金流入公司，在表上就会有正值表示，如果现金从公司流出，就以负值表示，它可以帮助企业得知自己目前的资金情况。

公司的现金流有三大类：

  • 营业活动现金流量（或称经营活动现金流量）
  • 投资活动现金流量
  • 募资活动现金流量

只要有上一期与这一期的利润表，加上期初和期末的两张资产负债表，就能推导出现金流量表。我们来看看现金流量表与利润表、资产负债表的关系：

期初现金 + 营业活动现金流量 + 投资活动现金流量 + 募资活动现金流量 = 期末现金

其中：营业活动现金流量可从利润表中获得，投资活动现金流量可从资产负债表的资产中（左边）获得，募资活动现金流量可从资产负债表的负债中（右边）获得。

三张表放在一起看，环环相扣，真的假不了，假的也真不了。

三大现金流量

我们以茅台为例，来看看茅台的现金流量表。
数据可在财报说，输入贵州茅台进行查询，在原始财报中的现金流量表部分，或者直接点击这里，后面关于茅台的数据均可在这里查到。

1. 营业活动现金流量

营业活动现金流量是以利润表的净利润为基础，将不是真正现金支出的折旧费用和分期摊销费用加回来，经过计算，得到营业活动现金流量。

一家企业到底是有钱入账还是赤字亏钱，通过这个指标一目了然。

茅台近五年基本上是逐年递增，2017年，企业扎扎实实赚进222亿现金。

如果一家企业的营业活动现金流量为负，可认为经营活动的流入少于流出，也就是说企业的销售收入不足以支付企业日常的人工开支和税务开支，即入不敷出。

2. 投资活动现金流量

一般情况下，投资活动的现金流量多为负值。

根据茅台近五年的数据来看均是负值（表中()是表示负值），表明企业为了持续经营，不断投资更多的机械、厂房等资产，以期为企业创造更多的收入。可理解为这家企业看好自己的行业和前景发展，所以持续加码，以满足未来成长所需。

这个数据来源于资产负债表的左边。

3. 筹资活动现金流量

  • 为正，表示筹资活动的流入大于流出，也就是企业不管采取哪种方式，是向股东借（向内）还是向银行（向外）借，反正是借到钱了。
  • 为负，表示公司对银行还款或者对股东分红了，或者两种情况都有。

这项数据来源于资产负债表的右边。

茅台近五年均为负值，2017年，还款、分红共计88.99亿元。

现金流量表的四大要点

  • 企业有没有真正的现金？
  • 获利含金量(OCF/NI)是否大于100？
  • 企业的OCF是否大于固定资产投资？
  • OCF和NI趋势是否一致？

其中：
OCF：营业活动现金流量
NI：净利润
补充：关于OCF的四大注意点

四、快收慢付

快收就是缩短平均收现天数，尽快收回资金，缩短做生意的周期；慢付就是应付账款慢慢支付，拉长应付账款付款天数。

还记得做生意的完整周期吗？
$$做生意的完整周期 = 存货周转天数 + 平均收现天数 = 应付账款付款天数 + 缺钱天数$$

我们下面来看一个极简案例，来深刻的理解一下利润与现金的错位。
小明大学毕业想创业，准备做倒卖图书的生意，家里人给予1000元创业资金支持。假设：

  • 一月份收入1000元；
  • 后面每个月都比上个月多50%的营业额，毛利40%；
  • 固定房租500元，无其他成本；
  • 上游厂家给他30天月结的优惠；
  • 他给下游60天结账的条件。

我们回顾一下之前的内容：
毛利：
$$毛利 = 营业收入 - 营业成本$$

营业利润：
$$营业利润 = 毛利 - 营业费用$$

由此，我们可以算出一季度的利润表如下：

1月2月3月备注
收入100015002250每个月收入增长50%
成本6009001350假设为60%
毛利400600900毛利 = 收入 - 成本
费用500500500每月房租
净利润-100100400净利润 = 毛利 - 费用

我们再来看看他的现金流量表：
没有投资、筹资行为，所以只有营业活动现金流量，上面说了，他给上游30天付款，他给下游60天结账，所以：

  1. 上游付款时间：一月份成本600元二月份付，二月份成本900元三月份付，以此类推。
  2. 收入进账时间：一月份收入1000元要等到三月份才能变成现金打入账户，二月份收入1500元要等到四月份才能进账，以此类推

由此，我们可以算出一季度的现金流量表如下：

1月2月3月备注
期初现金1000500-600本月期初 = 上月期末
收入进账100060天后下游结算
成本支出60090030天付款给上游厂家
费用支付500500500每月房租
期末现金500-600-1000

利润表是好看，但从现金流量表来看，三个月里没有实打实的现金进账，现金缺口愈演愈烈。这是个极简例子，如果我们把单位元换成万元，亿元呢？

所以说，利润表只是个推估的概念，要结合现金流量表一起看，综合立体的来判断。

我们想个办法来改善一下。

前面的假设条件“给上游30天付款，给下游60天结账”换成“给上游60天付款，给下游30天结账”，也就是说一月份的收入二月份到账，一月份的成本三月份才支付，重新来算一下现金流量表：

1月2月3月备注
期初现金10005001000本月期初 = 上月期末
收入进账1000150030天后下游结算
成本支出60060天付款给上游厂家
费用支付500500500每月房租
期末现金50010001400

可以看到，仅仅改变了收付款条件，现金流就得到了改善。

快收慢付理论上很容易理解，实际上如果长期拖付“应付账款”，上游供应商可能就不会跟你合作了，所以这当中需要自己去斟酌和谈判了。

三表合一来看公司的商业逻辑

公司的财务信息与经营特征之间的相互印证，是阅读财报中的一个重要的思维方式，也就是说，我们要观察公司的盈利模式与现金流、利润、资产等是否匹配。因此，我们基于财务信息对公司进行判断时，需要先确保数据背后的商业逻辑是否讲得通。

近几年来，我们经常看到一些现金流为负的公司也能成功上市，为什么？我们要关注现金流的好坏，更需要关注现金流结构。比如科蓝软件，它现在的现金流压力就不小，我们能立马评判它就是不好的企业么？仔细观察一下，科蓝软件眼下产生的现金不能养活自己，但有足够的外部输血，能融到资，另一方面，科蓝软件的客户主要是银行，钱大概率是能收回来的，只要未来现金流回流，就能养活自己了。

大部分的初创型公司的现金结构都会有科蓝软件这种情况，经营活动现金流和投资活动现金流都是负的，只有筹资活动是正的，这很大程度上是与公司的生命周期息息相关的，对于科蓝软件这类公司，我们与其关注现金流是否良好，更需关注现金结构是否合理。但是，对于衰退期的公司，当经营活动现金流为负时，就要非常小心了。
